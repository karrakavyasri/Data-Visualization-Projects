# Store Sales Visualization

## Description
This project visualizes a store’s sales data using Python's Matplotlib library.  
It shows monthly sales trends, product-wise sales, and regional sales all in one screen.

## Dataset
- **Months:** Jan → Dec
- **Monthly Sales:** Revenue for each month (in USD)
- **Products:** Laptops, Mobiles, Tablets, Accessories
- **Product Sales:** Total revenue per product
- **Regions:** North, South, East, West
- **Regional Sales:** Total revenue per region

> All data is sample data and can be replaced with real store data.

## Visualizations
1. **Line Chart – Monthly Sales**: Shows month-wise sales trend
2. **Bar Chart – Product Sales**: Compares revenue of different products
3. **Pie Chart – Regional Sales**: Shows percentage distribution of sales across regions
